# ls
Life simulation on Python
Симуляция жизни на языке программирования Python 3.7.2

# Как запустить?
1. Установите Python 3.*
2. Скачайте файл main.py
3. Откройте cmd и напишите команду **pip install numpy**
4. Откройте файл main.py
